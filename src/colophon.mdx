### Type

This site is mainly set in [_GT America_](https://www.grillitype.com/typeface/gt-america) by Grilli Type, a modern hybrid of American Gothic and European Neo-Grotesk typefaces.
$\TeX$ mathematical expressions are set with $\KaTeX$ in _Computer Modern_.
Code is set in _IBM Plex Mono_ by IBM.
